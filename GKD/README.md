### 通知

### 使用前须知

    本仓库仅为本人学习研究的成果。

    使用第三方订阅需要执行以下操作之一

    关闭（或删除）默认订阅

    将所有第三方订阅长按拖动到默认订阅之前


### 更新策略

此后以同步AIsouler订阅更新为主，夹带个人规则更新。

同时随时面临不定时、不定长的停更风险（若停更，停更至多半年）。

更新时间不固定。
